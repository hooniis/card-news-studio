4월 2일 아침,
파라다이스 호텔 부산 로비.

"혼자보다

## 카드 3 (본문) — 경험의 흐름

**유형**: 본문 (이미지 포함)

**헤드라인**:
```
달리고, 스파에 몸 담그고,
바다를 보며 쉬는 프로그램 🌊
```

**본문**:
```
러닝을 마친 뒤
4층 씨메르 스파로 향했어요.

긴장했던 근육이 서서히 풀렸어요.

통유리 너머로
해운대 앞바다가 펼쳐졌어요.
```

**프로세스 흐름** (아이콘 또는 화살표로 표현):
```
1. 달리고 🏃
   ↓
2. 씻고 💧
   ↓
3. 쉬는 🌊
```

**핵심 메시지 박스**:
```
여행의 목적이 바뀌고 있어요
"어디를 가느냐" → "어떻게 회복하느냐"
```

**강조 키워드**:
- 달리고, 씻고, 쉬는
- 어떻게 회복하느냐

**이모지**: 🌊 (헤드라인), 🏃 💧 🌊 (프로세스)

**이미지**: 뉴스 이미지 2번 (파라다이스 부산 씨메르 스파)

**출처**: 사진=신용현 기자, 출처: 한국경제

## 카드 4 (본문) — 트렌드 배경

**유형**: 본문 (데이터 중심)

**헤드라인**:
```
러닝 인구 1000만 시대,
여행의 목적이 바뀌고 있어요 📊
```

**본문**:
```
국내 러닝 인구
약 1,000만 명 돌파
(문화체육관광부 기준)

서울에만
100개 이상 러닝 크루 활동 중

2030세대가 주도하는 러닝 열풍,
단순 스포츠를 넘어
문화 코드로 자리잡았어요
```

**팩트 박스** (accent 핑크 배경):
```
런 클럽 문화 확산
+
호텔 피트니스 프로그램 다양화
=
새로운 여행 트렌드
```

**강조 키워드**:
- 1,000만 명 (크게 강조, 60~72px)
- 100개 이상
- 2030세대

**이모지**: 📊 (헤드라인)

**출처**: 출처: 한국경제 / 신용현 기자

## 카드 5 (본문) — 개념 정의

**유형**: 본문 (개념 강조)

**헤드라인**:
```
'소셜 웰니스'
함께 움직이고, 각자 쉬는
새로운 웰니스 💚
```

**본문 (Before/After 비교)**:
```
📍 기존 웰니스
스파·명상 중심
혼자 회복하는 여행

📍 새로운 웰니스
함께 땀 흘리고
이후 각자 쉬는
'소셜 웰니스'
```

**부연 설명 박스**:
```
사회적 관계가
건강과 수명에 중요한 영향을 미쳐요

웰니스 초점이 바뀌고 있어요
"무엇을 하느냐" → "누구와 어떻게 연결될 것인가"
```

**강조 키워드**:
- 소셜 웰니스 (48~56px Bold, 중앙 정렬)
- 함께 움직이고, 각자 쉬는

**이모지**: 💚 (헤드라인), 📍 (구분자)

**출처**: 출처: 한국경제 / 신용현 기자

⭐ 소노 비발디파크
'소노 런트립 180K' 패키지
나이트 런 + 모닝 런

**강조 키워드**:
- 파라다이스호텔 부산

- 소노 비발디파크
- (각 브랜드명 Bold 처리)
