# 카드뉴스 품질 검증 보고서

## 종합 판정: PASS (조건부)

전체적으로 카피-디자인 정합성, 컬러 팔레트, 레이아웃 사양, 스토리라인 흐름 모두 우수하다. 일부 이모지 선택이 카피 문서와 다르나, HTML에서 사용된 이모지가 맥락상 더 적절하여 실질적 품질 문제는 아니다. 아래 FIX 항목 1건을 반영하면 최종 PASS 처리 가능하다.

---

## 카드별 검증

### 카드 1 (커버)
- **판정: PASS**
- 카피 정합성: 일치. 헤드라인 "스타벅스가 야구장 갑니다", 서브헤드라인 "KBO x 스타벅스, 최초 콜라보 출시" 모두 HTML과 동일.
- 타이포그래피: 헤드라인 80px (기준 72~84px 충족), 서브헤드라인 30px -- 적정.
- 컬러: Primary #00704A 그라데이션 배경, 화이트 텍스트 -- 전략 팔레트 준수. 명도 대비 양호.
- 레이아웃: 1080x1080, Safe area 80px (top/left/right/bottom 모두 80px) -- 준수.
- 폰트: Pretendard CDN 로드 -- 준수.
- 브랜드: @요즘뭔일 좌하단, 1/6 우하단 -- 준수.
- 이모지: 2개 (커피, 야구공) -- 기준 충족.

### 카드 2 (본문)
- **판정: PASS**
- 카피 정합성: 일치. 헤드라인 "이거 진짜 처음이에요", 본문 3줄, 강조 텍스트 "국내 스포츠 최초 공식 협업" 모두 HTML에 정확히 반영.
- 타이포그래피: 제목 56px (기준 52~60px 충족), 본문 38px (기준 36~40px 충족).
- 컬러: 배경 #FFF8F0 (Accent), 텍스트 #1A1A2E (Text), 강조박스 #00704A (Primary) -- 전략 팔레트 준수.
- 레이아웃: Safe area 80px -- 준수.
- 브랜드: @요즘뭔일, 2/6 -- 준수.
- 이모지: 1개 (별) -- 카피 문서에는 star2 + handshake 2개 명시되어 있으나 HTML에는 별 1개만 사용. 1~2개 기준 내이므로 문제없음.
- 비고: 카피 문서의 handshake 이모지 누락은 디자인 판단으로 수용 가능. 제목에 이모지가 1개인 것이 가독성 측면에서 오히려 낫다.

### 카드 3 (본문)
- **판정: PASS**
- 카피 정합성: 일치. 헤드라인 "내 팀 굿즈가 스타벅스에서?", 본문 텍스트, 강조 텍스트 "구단별 전용 MD 출시" 모두 정확히 반영.
- 타이포그래피: 제목 56px, 본문 38px -- 기준 충족.
- 컬러: 배경 #FFF8F0, 강조박스 #E03131 (Secondary) -- 카드 2와 차별화하면서 전략 팔레트 준수.
- 레이아웃: Safe area 80px -- 준수.
- 브랜드: @요즘뭔일, 3/6 -- 준수.
- 이모지: 1개 (불꽃) -- 카피 문서에는 tumbler_glass + fire 2개 명시, HTML에는 fire 1개. 기준 내.

### 카드 4 (본문 + 이미지)
- **판정: PASS**
- 카피 정합성: 일치. 헤드라인 "실물 미쳤어요", 본문 "콜라보 음료, 푸드도 함께 출시!" 정확히 반영.
- 이미지: `<img src="../starbucks.jpg">` -- 경로 ../starbucks.jpg 정확. alt 텍스트 "스타벅스 x KBO 콜라보 MD 실물" 포함.
- 타이포그래피: 제목 56px, 본문 36px -- 기준 충족.
- 컬러: 배경 #FFF8F0, 디바이더 #00704A -- 준수.
- 레이아웃: Safe area 80px, 이미지 컨테이너 border-radius 20px -- 준수. 이미지가 중앙~하단에 크게 배치되어 전략 의도 충족.
- 브랜드: @요즘뭔일, 4/6 -- 준수.
- 이모지: 1개 (카메라) -- 기준 충족.

### 카드 5 (본문)
- **판정: PASS**
- 카피 정합성: 일치. 헤드라인 "언제부터 살 수 있어?", 체크리스트 3개 항목(출시일, 구매처, 한정판) 모두 정확히 반영. 강조 텍스트 "3월 27일 순차 출시"가 highlight-box에 반영.
- 타이포그래피: 제목 56px, 체크리스트 38px, 강조 날짜 52px -- 기준 충족.
- 컬러: highlight-box에 #00704A 그라데이션 + #FFD43B (Highlight/골드 옐로우)로 날짜 강조 -- 전략의 "골드 옐로우 하이라이트로 핵심 날짜 표시" 의도 정확히 반영.
- 레이아웃: Safe area 80px, 체크리스트 스타일 -- 전략의 "체크리스트 or 캘린더 스타일" 의도 충족.
- 브랜드: @요즘뭔일, 5/6 -- 준수.
- 이모지: 1개 (달력) + 체크마크(기능적 아이콘) -- 기준 충족.

### 카드 6 (CTA)
- **판정: FIX**
- 카피 정합성: 헤드라인 "야구 좋아하는 친구한테 알려줘요!" -- 일치. CTA 버튼 "저장 + 공유하기" -- 일치. 부가 텍스트 "시즌 개막 전에 꼭 챙겨두세요" -- 일치. 댓글 유도 "응원하는 구단 댓글로 알려줘요!" -- 일치.
- 타이포그래피: 헤드라인 56px, CTA 버튼 38px, 부가텍스트 30px -- 적정.
- 컬러: 스타벅스 그린 그라데이션 배경 (#00704A ~ #34d399) -- 전략 의도 충족.
- 레이아웃: Safe area 80px -- 준수.
- 브랜드: @요즘뭔일, 6/6 -- 준수.
- 이모지: 2개 (저장 아이콘, 말풍선) -- 기준 충족.
- **FIX 사유**: 전략에서 "@요즘뭔일 팔로우 유도 문구 하단 배치"를 명시하고 있으나, 현재 HTML에는 팔로우 유도 문구가 없다. @요즘뭔일 계정명은 있으나, "팔로우하고 소식 받아보세요" 같은 명시적 팔로우 유도 문구가 필요하다.
- 수정 지시: comment-cta 아래에 팔로우 유도 문구 추가. 예: `<p class="follow-cta">@요즘뭔일 팔로우하고 소식 받아보세요</p>` (font-size: 26px, color: rgba(255,255,255,0.7) 정도)

## 전체 이슈

### 디자인 일관성
- 6장 모두 동일한 CSS 변수 체계(:root)를 사용하여 컬러 일관성 확보 -- 양호.
- 본문 카드(2~5)는 #FFF8F0 배경 + #1A1A2E 텍스트로 통일 -- 양호.
- 커버(카드 1)와 CTA(카드 6)는 그린 계열 그라데이션 배경으로 시작과 끝을 시각적으로 묶어줌 -- 양호.
- 모든 카드에 동일한 Pretendard 폰트 CDN 로드 -- 양호.
- @요즘뭔일 (좌하단 18px) + 페이지 인디케이터 (우하단 24px) 6장 모두 동일 위치/크기 -- 양호.

### 스토리 흐름
- 전략의 감정선 "오 대박 -> 이런 거였어 -> 이것도 있어 -> 언제 나와? -> 빨리 가자!" 순서와 카드 흐름이 정확히 일치한다.
- 카드 1(후킹) -> 2(뉴스 전달) -> 3(상세1: MD) -> 4(상세2: 실물) -> 5(정리) -> 6(CTA) -- 전략 스토리라인과 완벽히 일치.

### 톤앤매너
- "이거 진짜 처음이에요", "내 팀 굿즈가 스타벅스에서?", "실물 미쳤어요", "언제부터 살 수 있어?" -- 친근한 반말 + 감탄사 톤으로 전략 의도 충족.
- 문장 길이 짧고 임팩트 있음 -- 전략의 "한 문장 15자 내외" 권장 충족.
