# 카드뉴스 카피 — 단백질 몰빵

## 카드별 카피

### 카드 1 (커버)
- 제목: 단백질 백날 먹어도 근육 안 생기는 이유
- 부제: 운동하고 고기 먹는데 왜 안 되지?
- 출처: 사진=게티이미지뱅크

### 카드 2 (문제 제기)
- 제목: 한국인의 단백질 습관
- 핵심 수치: 하루 단백질의 저녁 쏠림 비율 70% 이상
- 아침: 빵·시리얼 위주, 단백질 거의 0
- 저녁: 삼겹살·회식으로 60~80g 폭탄
- 강조: 이게 바로 '단백질 몰빵'입니다

### 카드 3 (메커니즘)
- 제목: 몸에선 이런 일이 벌어져요
- 플로우: 저녁에 단백질 60~80g 폭식 → 1끼 흡수 한계 20~30g 초과 → 남은 단백질 근육 합성 불가 → 지방 축적 + 간·신장 부담↑
- 경고: 비싼 고기 먹고 근육 대신 지방만 얻는 격!

### 카드 4 (회식 문제)
- 제목: 회식 고기는 더 문제예요
- 공식: 술 + 고기 = 밑 빠진 독에 물 붓기
- 문제1: 알코올이 근육 합성 호르몬 억제 + 단백질 대사 방해
- 문제2: 후식 냉면·볶음밥 → 혈당 스파이크 → 단백질이 지방으로 전환

### 카드 5 (해결법)
- 제목: 이렇게 나눠 먹으세요
- 핵심: 단백질 1:1:1 분산 섭취 (근육 합성 효율 25% 향상!)
- 아침 20~30g: 달걀 2개 + 우유
- 점심 20~30g: 닭가슴살 1덩이
- 저녁 20~30g: 손바닥 크기 고기
- 팁: 아침 달걀 1개가 저녁 고기 3인분보다 근육에 훨씬 효과적

### 카드 6 (CTA)
- 제목: 헬스하는 친구에게 공유해주세요!
- CTA 버튼: 저장하고 공유하기
- 질문: 단백질 어떻게 챙기세요?
