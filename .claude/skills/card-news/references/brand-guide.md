# 브랜드 가이드 — @요즘뭔일

모든 에이전트가 참조하는 브랜드 아이덴티티 가이드. 여러 편의 카드뉴스가 하나의 계정에서 일관되게 보이도록 통일된 기준을 제공한다.

## 계정 정보

- **계정명**: @요즘뭔일
- **슬로건**: 요즘 세상 무슨 일? 3분이면 충분해요
- **성격**: 생활정보 + 종합 뉴스 큐레이션
- **타겟**: 2030 직장인

## 카피 톤앤매너

### 어조 원칙
친구한테 카톡으로 설명하듯 쓴다. 뉴스를 전달하되 딱딱하지 않게, 쉽게.

| 항목 | 기준 |
|------|------|
| 종결어미 | "~이에요", "~거예요", "~했어요" |
| 호칭 | 독자를 직접 부르지 않음. 필요시 "우리" |
| 전문용어 | 풀어서 설명. 첫 등장 시 한 줄 부연 |
| 이모지 | 카드당 1~2개. 제목이나 핵심 포인트 옆에 배치. 본문 중간에 남발 금지 |
| 문장 길이 | 한 문장 20자 이내. 짧게, 끊어서 |
| 유머 | 가볍게 허용. 자극적이거나 비꼬는 톤은 금지 |

### 예시

```
❌ "원유 공급 불안정으로 인해 나프타 수급에 차질이 빚어질 가능성이 제기되고 있습니다."
✅ "전쟁 때문에 원유가 불안해요. 원유에서 나오는 나프타, 이게 비닐 원료거든요."

❌ "전주 대비 321.9% 증가한 것으로 나타났다."
✅ "지난주보다 3배 넘게 팔렸어요. 321.9%↑"
```

### CTA 패턴
마지막 카드의 CTA는 항상 다음 구조를 따른다:
1. 행동 유도 한 줄 (저장/공유)
2. 댓글 유도 질문 1개
3. 계정명 노출

```
"도움이 됐다면 저장 & 공유 부탁해요!
여러분 동네는 어떤가요? 댓글로 알려주세요 💬
@요즘뭔일"
```

## 비주얼 아이덴티티

### 컬러 전략: 주제별 변경
매 편마다 주제에 맞는 컬러 팔레트를 선택한다. 단, 아래 고정 요소는 모든 편에서 동일하게 유지한다.

**고정 요소:**
| 요소 | 값 | 용도 |
|------|---|------|
| 계정명 텍스트 | `#FFFFFF` 또는 `#1E293B` (배경 대비) | 하단 계정명 |
| 페이지 인디케이터 | `#64748B` (밝은 배경) 또는 `rgba(255,255,255,0.6)` (어두운 배경) | 하단 우측 |
| 배경 회색 (body) | `#E2E8F0` | 브라우저 배경 |

**주제별 팔레트 선택 기준:**
| 주제 | Primary | Accent | 커버 배경 |
|------|---------|--------|----------|
| 경제/금융 | `#0F766E` | `#EAB308` | 다크 그래디언트 |
| 사회/이슈 | `#DC2626` | `#F59E0B` | 다크 그래디언트 |
| IT/테크 | `#2563EB` | `#8B5CF6` | 다크 그래디언트 |
| 건강/웰빙 | `#059669` | `#F472B6` | 라이트 그래디언트 |
| 생활/소비 | `#7C3AED` | `#34D399` | 다크 그래디언트 |
| 부동산/주거 | `#1D4ED8` | `#F97316` | 다크 그래디언트 |
| 기타 | `#334155` | `#38BDF8` | 다크 그래디언트 |

### 커버 카드 레이아웃 (통일)

모든 편의 커버 카드(카드 1)는 동일한 레이아웃을 따른다:

```
┌──────────────────────────┐
│  (80px 여백)              │
│                          │
│  [주제 이모지] 56px       │
│                          │
│  [헤드라인]               │
│  56~72px, ExtraBold      │
│  중앙 정렬                │
│                          │
│  ─── divider ───         │
│                          │
│  [서브 헤드라인]          │
│  28~32px, Regular        │
│  중앙 정렬                │
│                          │
│                          │
│  @요즘뭔일     1/N       │
│  (좌하단)     (우하단)    │
└──────────────────────────┘
```

**커버 카드 CSS 고정 규칙:**
- 배경: 어두운 그래디언트 (주제별 primary → dark 방향)
- 텍스트: 흰색 (`#FFFFFF`)
- 정렬: 전체 중앙 정렬 (`text-align: center`)
- 이모지: 상단에 1개, 56px
- divider: 중앙, 60px 너비, 흰색 반투명 (`rgba(255,255,255,0.4)`)

### 본문 카드 레이아웃
본문 카드는 주제에 따라 유연하게 변형 가능하되, 다음 고정 요소를 포함한다:
- 좌하단: `@요즘뭔일` 계정명
- 우하단: 페이지 인디케이터 (`N/총수`)
- Safe area: 80px

### 계정명 표시 CSS

```css
.account-name {
  position: absolute;
  bottom: 40px;
  left: 80px;
  font-size: 18px;
  font-weight: 600;
  letter-spacing: -0.02em;
}

/* 밝은 배경 카드 */
.account-name.light { color: #64748B; }

/* 어두운 배경 카드 */
.account-name.dark { color: rgba(255, 255, 255, 0.6); }
```

### CTA 카드 레이아웃 (통일)

모든 편의 CTA 카드(마지막 카드)도 통일된 레이아웃을 따른다:

```
┌──────────────────────────┐
│                          │
│  [행동 유도 헤드라인]     │
│  40~48px, Bold           │
│                          │
│  ─── divider ───         │
│                          │
│  [CTA 버튼]              │
│  저장하고 공유하기        │
│                          │
│  [댓글 유도 질문]         │
│  24~28px                 │
│                          │
│  @요즘뭔일     N/N       │
└──────────────────────────┘
```

**CTA 카드 CSS 고정 규칙:**
- 배경: primary 컬러 기반 그래디언트 (밝은 톤)
- CTA 버튼: 흰색 배경, primary 텍스트, border-radius 12px
- 톤: 커버/본문의 긴장감에서 차분하게 마무리

## 해시태그 전략

모든 편에 공통으로 포함하는 고정 태그 + 주제별 태그:

**고정 태그 (매 편 포함):**
```
#요즘뭔일 #카드뉴스 #뉴스정리 #시사상식 #생활정보
```

**주제별 태그:** 7~10개 추가 (총 12~15개)

[CTA — 저장/공유/댓글 유도]
