# SNS 플랫폼별 카드뉴스 사양

content-strategist와 card-designer 에이전트가 참조하는 플랫폼별 이미지 사양과 가이드라인.

## 플랫폼별 이미지 사양

| 플랫폼 | 포맷 | 크기 (px) | 비율 | 최대 장수 | 파일 형식 |
|--------|------|----------|------|----------|----------|
| **Instagram 피드** | 정사각형 | 1080 x 1080 | 1:1 | 10장 | JPG/PNG |
| **Instagram 피드** | 세로형 | 1080 x 1350 | 4:5 | 10장 | JPG/PNG |
| **Instagram 스토리** | 풀스크린 | 1080 x 1920 | 9:16 | — | JPG/PNG |
| **Facebook** | 정사각형 | 1080 x 1080 | 1:1 | 10장 | JPG/PNG |
| **LinkedIn** | 정사각형 | 1080 x 1080 | 1:1 | 9장 | JPG/PNG |
| **LinkedIn** | 세로형 | 1080 x 1350 | 4:5 | 9장 | JPG/PNG |
| **X (Twitter)** | 가로형 | 1200 x 675 | 16:9 | 4장 | JPG/PNG |
| **카카오톡 채널** | 정사각형 | 800 x 800 | 1:1 | 10장 | JPG/PNG |
| **네이버 블로그** | 가로형 | 1200 x 900 | 4:3 | 제한없음 | JPG/PNG |

## 기본값

플랫폼이 지정되지 않으면 **Instagram 피드 정사각형 (1080x1080)**을 기본으로 사용한다. 가장 범용적이며 다른 플랫폼에서도 무난하게 표시된다.

## 플랫폼별 고려사항

### Instagram
- 피드: 하단에 좋아요/댓글 UI가 오버레이됨. 하단 120px은 안전 영역 확보
- 캐러셀: 넘기기 동작으로 카드 전환. 각 카드 오른쪽 가장자리에 "넘기세요" 암시 가능
- 해시태그: 최대 30개, 10~15개가 효과적
- 캡션: 최대 2,200자

### LinkedIn
- 전문적인 톤 유지. 밈, 이모지 과다 사용 자제
- 캐러셀(문서 포스트): PDF 업로드 형태로 최대 300페이지까지 가능하나 9장 이내 권장
- 해시태그: 3~5개가 적정

### Facebook
- 텍스트 비율 20% 이하 권장 (광고 기준이지만 유기 포스트에도 적용하면 도달률 향상)
- 밝고 선명한 이미지가 알고리즘에 유리

### X (Twitter)
- 이미지 4장 제한. 핵심 메시지만 압축
- 세로형은 잘려서 표시됨. 가로형 권장
- 텍스트 크기를 크게 — 타임라인에서 썸네일로 보이므로

## HTML 크기 적용

card-designer는 플랫폼에 따라 `.card`의 width/height를 조정한다:

```css
/* Instagram 정사각형 (기본) */
.card { width: 1080px; height: 1080px; }

/* Instagram 세로형 */
.card { width: 1080px; height: 1350px; }

/* X (Twitter) */
.card { width: 1200px; height: 675px; }
```
